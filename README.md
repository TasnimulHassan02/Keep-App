# Keep - Note-Taking App

**Keep** is a modern, interactive, and user-friendly **note-taking application** built with **React**. It allows users to quickly create, edit, delete, and organize notes, providing a digital alternative to physical notebooks. The app features a **clean UI inspired by Google Keep**, making note management intuitive and visually appealing.

## Features

- **Add Notes**: Create notes with a title and content.
- **Edit Notes**: Easily update existing notes.
- **Delete Notes**: Remove notes instantly with a single click.
- **Dynamic Layout**: Responsive grid layout that adapts to different screen sizes.
- **Material-UI Components**: Uses MUI v5 icons and buttons for modern styling.

# Technologies Used

- **React** – Frontend framework

- **Material-UI** – Icons and UI components
  
- **HTML** – Structure layout

- **CSS** – Styling and responsive layout

- **JavaScript** - Programming Language
